# Trilium-Table-Count-and-Sum
Trilium表格计数与求和插件（针对0.102.2版本，其他版本未测试）

使用方法：

1.导入到Trilium（取消选择安全导入）。

2.鼠标选中待计算区域的第一个单元格。

3.按下shift按键，鼠标点击待计算区域的最后一个单元格，松开shift按键。

4.右下角显示计数与求和弹窗。

5.进行任何取消选中单元格的操作，弹窗消失。
